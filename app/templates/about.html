About Dithering

Dithering is a technique used to achieve illusion of „colour depth” with limited colour palette.

On this site there are implemented two types of dithering algorithms: ordered dithering and error diffusion dithering.

First uses threshold map to determine output colour on pixel by pixel basis. Size of this map should be significantly smaller than size of the picture. Used maps (Bayer, Halftone) differ by size and and values of thresholds.

Second uses fixed threshold of 50% but distributes quantification error onto neighbouring pixels using kernel which describes what fraction of error should be added to specific pixels. Used error diffusion (Floyd–Steinberg; Jarvis, Judice & Ninke) algorithms differ by size of kernel.
